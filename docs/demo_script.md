# 1 分钟演示脚本

## 0-10 秒：打开页面

打开 `http://127.0.0.1:5173`。展示左侧“小熙”角色区域、右侧聊天窗口、状态面板、记忆面板和待办面板。

旁白：这是一个生活陪伴型虚拟角色 Agent MVP，角色名叫小熙。当前使用占位 Live2DViewer，后续可替换真实 Live2D 模型。

## 10-20 秒：角色自我介绍

用户输入：

```text
你好呀，你是谁？
```

预期效果：小熙以原创角色身份介绍自己，状态面板显示 `emotion` 和 `skills_used`。

## 20-30 秒：情绪陪伴

用户输入：

```text
我今天完全不想动，感觉已经开摆了。
```

预期效果：小熙用轻微网络语感接住用户情绪，给出最小行动建议。角色状态切换到 `sad` 或 `thinking`。

## 30-42 秒：长期记忆

用户输入：

```text
记住我最近在做 Live2D 虚拟陪伴 Agent。
```

预期效果：后端写入 `backend/storage/memory.json`，状态面板显示 `memory_action: write`，记忆面板刷新后出现新记忆。

继续输入：

```text
你还记得我最近在做什么吗？
```

预期效果：小熙读取长期记忆并回答，状态面板显示 `memory_action: read`。

## 42-55 秒：工具调用

用户输入：

```text
帮我安排今晚两小时完成报告。
```

预期效果：Planner 决定调用 `study_plan` 工具，后端真实执行工具并生成计划，状态面板显示 `tool_used: study_plan`。

## 55-60 秒：总结

展示状态面板。

旁白：这个 MVP 包含原创人格、两阶段 LLM 决策、长期记忆、工具调用、mock 模式和 emotion 驱动的前端角色状态，适合课程报告和视频演示。
