# Course Report

## Instructions
- Organize reports around objective, method, implementation, results, analysis, and conclusion.
- Preserve user-provided facts and avoid inventing data.
- Make technical explanations clear enough for coursework grading.
- Suggest figures, tables, or appendices when they improve readability.
